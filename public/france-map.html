Aide · sources · crédits

Utilisation

V1 — 13 régions métropolitaines. La fiche (panneau de droite) est repliée par défaut ; ouvrez-la avec le bandeau « Fiche » ou en cliquant une région.

Contours des régions — coût de la vie, climat et villes : la fiche à droite résume la région sélectionnée.

Bike parks descente montagne : données OSM (mtb:type=downhill, mtb:lift=yes) — wiki OSM · KelBikePark.

Couches : une pastille par type. Motocross : terrains OpenStreetMap (sport=motocross) en ambre ; annuaire FFM (circuits tout-terrain) en violet, lien « FFM » dans l’infobulle — carte officielle : carte.ffmoto.org. Yelloh! : taille des points selon la note satisfaction. Sandaya : taille légère selon les étoiles. Après clic sur une région, les villes de la fiche peuvent apparaître en points indigo.

Données & actualisation

Géolocalisation des lieux : OpenStreetMap (Overpass).

Campings Yelloh! Village : France via l’API du site (searchCampings) — yellohvillage.fr · npm run fetch:fr-yelloh-official.

Campings Sandaya (France métropolitaine & Corse) : page liste — sandaya.fr/nos-campings · npm run fetch:fr-sandaya.

Sources hors OSM (annuaires, fédération, presse, stations) : liste JSON · KelBikePark · Sites VTT FFC · Owlaps.

Technique

Carte : Leaflet + D3. Contours des régions : Etalab.
